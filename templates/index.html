Model Details

Accuracy 86.73%
Input Features 7 — Age, Income, Role, Overtime, Travel, Satisfaction, Tenure
Output Classes Binary — Attrition: Yes / No

The IBM HR Analytics dataset contains records for 1,470 employees across 35 features. This implementation uses 7 key predictors selected through feature importance analysis. The Random Forest ensemble averages predictions across multiple decision trees to reduce overfitting and improve generalization.

03

About

What this project is and why it was built.

AttritionAI is an AIML course project applying supervised machine learning to HR analytics. The goal is to give organizations a data-backed signal when identifying employees who may be considering leaving — allowing HR teams to intervene proactively.

Built with Python's Scikit-Learn and served via Flask. No external APIs or cloud ML services are used — the classifier runs entirely server-side.

Language Python 3
Framework Flask
ML Library Scikit-Learn
Dataset IBM HR Analytics
Course AIML Project
