# Employee Attrition Prediction AI

This project predicts whether an employee will leave a company using Machine Learning.

## Technologies Used

- Python
- Flask
- Random Forest
- Scikit-Learn
- HTML/CSS

## Dataset

IBM HR Analytics Dataset

## Features

- Age
- MonthlyIncome
- JobRole
- OverTime
- BusinessTravel
- JobSatisfaction
- YearsAtCompany

## Model Accuracy

86.73%

## How to Run

Install dependencies:

pip install flask pandas scikit-learn joblib

Run:

python train_model.py

Then:

python app.py

Open:

http://127.0.0.1:5000/
